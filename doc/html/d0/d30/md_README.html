La génération de PDF

Lorsque l'on clique sur "générer un pdf", l'application va essayer de télécharger le pdf dans 3 endroits différents. Il est possible que l'un des 3 téléchargement échoue (message d'erreur dans ce cas), mais au moins un devrait fonctionner. Le message de confirmation peut être caché sous un message d'erreur car les notifications s'affichent par ordre anti-chronologique (dernier message généré >> affiché en premier).
